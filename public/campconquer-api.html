/games

all the games

/games get

get

get all the games

get /games

get all the games

  • Response

HTTP status code 200

Body

Media type: application/json

Type: array of Game

Items: Game

  • id: required (number)
  • pieces: required (array of Piece)

    state of all pieces at the beginning of the game

    Items: Piece

    • team: required (string)
    • player_name: required (string)

      for display purposes during gameplay

    • player_id: required (integer)

      readonly - not valid during update

    • body_type: required (gender_neutral_1 gender_neutral_2 male female)
    • face: required (string)
    • hair: required (string)
    • skin_color: required (string)
    • hair_color: required (string)
    • role: required (one of offense, defense)
    • path: required (array of Point)

      see Path

      Items: Point

      • x: required (number)
      • y: required (number)
    • speed: required (number)
    • health: required (integer)
    • range: required (number)

      distance of target within which a shot will be launched

    • health_bonus: required (number)
    • speed_bonus: required (integer)
    • range_bonus: required (number)
    • gear_owned: required (array of string)

      gear short name

    • gear_equipped: required (array of string)
    • ammo: required (array of string)

      list of ammo IDs (balloon, arrow, bomb)

  • locked: required (boolean)

    if true, game setup cannot be changed

  • winner: required (string)

    team name of winner, or null if a draw

  • moves: required (string)

    all the moves and throws, as a huge json string blob :-( -- not always present in output

  • match_length: required (integer)

    duration of game in seconds

  • player_outcomes: required (array of Outcome)

    once played, stats for each player of this game; should be in alphabetical order by team name when returned by server

    Items: Outcome

    • pickups: required (integer)

      number of times flag was picked up

    • throws: required (integer)

      number of balloons thrown

    • created: required (datetime)
    • captures: required (integer)

      number of times this team captured the flag (i.e. games this team won)

    • team: required (string)
    • flag_carry_distance: required (number)

      number of meters this team carried the flag

    • defend_mvp: required (boolean)

      was this player a top defender this game?

    • takedowns: required (integer)

      count of players on other teams who died at this team's (or player's) hand

    • attack_mvp: required (boolean)

      was this player a top attacker this game?

  • team_summaries: required (array of Summary)

    (read-only) once played, stats for each team for this game

    Items: Summary

    • created: required (datetime)
    • team: required (string)
    • captures: required (integer)

      number of times this team captured the flag (i.e. games this team won)

    • takedowns: required (integer)

      count of players on other teams who died at this team's (or player's) hand

    • throws: required (integer)

      number of balloons thrown

    • pickups: required (integer)

      number of times flag was picked up

    • flag_carry_distance: required (number)

      number of meters this team carried the flag

    • attack_mvps: required (array of integer)

      player ids (for team_summary only)

    • defend_mvps: required (array of integer)

      player ids (for team_summary only)

    • attack_mvp: required (number)

      number of times this player was a top attacker (for player_summary only)

    • defend_mvp: required (number)

      number of times this player was a top defender (for player_summary only)
